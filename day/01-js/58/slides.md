# 命令式与声明式
- 命令式编程（Imperative）
命令式编程的主要思想是关注计算机执行的步骤，即一步一步告诉计算机先做什么再做什么。

- 声明式编程 (Declarative)

声明式编程是以数据结构的形式来表达程序执行的逻辑。它的主要思想是告诉计算机应该做什么
声明式代码大多都是接近自然语言的。简洁优雅。


> React 也是声明式的，你只要描述你的 UI，接下来状态变化后 UI 如何更新，是 React 在运行时帮你处理的，而不是靠你自己去渲染和优化 diff 算法。
